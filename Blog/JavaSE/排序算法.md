## 排序算法

### 希尔排序

```java
public class Shell {
    public static void sort(Comparable[] a) {
        //升序排列
        int N = a.length;
        int h = 1;
        while(h < N/3) h = 3*h + 1;
        while(h>0) {
            for(int i = h; i < N; i++) {
                for(int j = i; j >=h && less(a[j], a[j-h]); j -= h) {
                    exch(a, j, j-h);
                }
            }
            h = h/3;
        }
    }
}
```
